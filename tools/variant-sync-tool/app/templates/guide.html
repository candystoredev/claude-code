How to Use

This tool syncs your Shopify product variants with a distributor/manufacturer product list. Upload both files, and it generates a Matrixify-ready Excel file you can import directly into Shopify.

1

Export from Shopify

Use the Matrixify app to export your products from Shopify as an Excel file. The export should include columns like Handle, Title, Option1 Value, Variant SKU, and Variant Barcode.

2

Get your distributor file

This is the product offerings spreadsheet from your distributor or manufacturer. The tool auto-detects columns like Product Name, SKU, Price, Flavor, and Size — no reformatting needed.

3

Upload & run sync

Upload both files on the home page. Optionally filter by manufacturer name if your distributor file has multiple brands. Click Run Sync.

4

Review results

The results page shows four categories:

  • Matched — Shopify variants that matched a distributor product (no action needed)
  • To Delete — Shopify variants not found in the distributor list (discontinued or removed)
  • To Add — Distributor products not in Shopify yet (new products to list)
  • Needs Review — Low-confidence matches that need your manual verification
5

Download & import

Download the generated Excel file. Each row has a Command column:

CommandMeaning
(blank)Matched variant — informational, no action
DELETERemove this variant from Shopify
NEWCreate a new product in Shopify
MERGEAdd as a new variant to an existing product
REVIEWLow-confidence match — verify before importing

Review the DELETE and REVIEW rows, then import via Matrixify.

How Matching Works

The tool uses a multi-pass matching strategy to pair Shopify variants with distributor products.

Pass 1: Deterministic matching

Fast, rule-based matching that runs without any AI calls:

  1. SKU match — Normalized SKU comparison (strips ND-prefixes, ignores case/punctuation). Highest confidence.
  2. Barcode/UPC match — Strips leading zeros and non-digit characters, then compares.
  3. Option-in-name match — Checks if the Shopify option value (e.g. "Bubble Gum") appears in the distributor product name. Handles accents, compound words, and uses specificity-based conflict resolution.

Pass 2: Claude AI fallback

Any variants unmatched after Pass 1 are sent to Claude (Haiku) for fuzzy matching. Claude evaluates flavor/variety name similarity with context awareness — it handles abbreviations ("Choc PB" = "Chocolate Peanut Butter"), alternate spellings, and naming conventions.

Matches above 85% confidence are auto-accepted. Matches between 50-85% are flagged for review. Below 50% are treated as unmatched.

Size scoping

Before matching, variants are scoped by product handle (each Shopify product matches independently). Distributor rows are filtered by unit size extracted from the product title and price fields (e.g. "5lb", "10oz"), so a 5lb product only matches against 5lb distributor offerings.

Safety nets

Before flagging a variant for deletion, the tool checks if its SKU exists anywhere in the distributor data. If so, it moves the variant to "Needs Review" instead of "Delete" — the product is still offered, just couldn't be matched by name.

Tech Stack

Backend

  • Python 3.12 — Runtime
  • FastAPI — Web framework
  • Jinja2 — HTML templating
  • Gunicorn + Uvicorn — Production server

Data Processing

  • pandas — DataFrame operations for parsing and matching
  • openpyxl — Excel file reading and writing

AI / Matching

  • Anthropic SDK — Claude API for fuzzy matching
  • Claude Haiku — Fast, cost-efficient model for flavor/variant matching

Infrastructure

  • Docker — Containerized deployment
  • Railway — Hosting platform
  • Doppler — Secrets management (API keys)

Column Auto-Detection

The tool auto-detects distributor file columns using pattern matching. Here's what it looks for:

FieldExample column names detected
Product NameProduct Name, Item Name, Description, Product Title
Flavor / VarietyFlavor, Flavour, Variety, Scent
Size / WeightSize, Weight, Net Weight, Pack Size
PricePrice, Cost, MSRP, Wholesale
SKUSKU, Item Number, Item #, Product Code
ManufacturerManufacturer, Brand, Vendor

Only Product Name is required. All other columns improve matching quality but are optional.
